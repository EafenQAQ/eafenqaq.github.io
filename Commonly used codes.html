img插入图片

<img src="" alt=""> 这段代码是HTML中的一个元素，用来嵌入图片。代码中的标签和属性解释如下：

  • <img>：这是HTML中的“image”标签，用于在网页上显示图像。
  • src（source的缩写）：指定图像文件的路径和文件名。这里的路径为空，意味着没有指定图像源。
  • alt（alternative text的缩写）：提供图像的替代文本，这在图像无法显示时非常有用，也有助于搜索引擎优化（SEO）和为视障用户提供信息。
  • width="" height=“” 可以用来定义宽高

这个标签的功能是向网页中嵌入一张图片。由于没有提供src属性的值，所以实际上这个标签不会显示任何图片。如果你想看到图片效果，需要在src属性中填入有效的图片URL或本地文件路径。

CSS 中常用的属性包括：

  1. Color 相关属性：

  2. 字体相关属性：

  8. border-style：设置边框的样式，如实线、虚线、点线等。 border-style属性用于设置边框的样式，可以使用以下值：

  9. none：无边框。

Javascript

var variable 声明一个变量，用于声明全局变量或函数作用域内的变量 let let 是 JavaScript 中用于声明变量的关键字之一，它是 ES6（ECMAScript 2015）引入的新特性之一。相比于旧版本的 var 关键字，let 具有更好的作用域规则和变量声明方式。 let 是 JavaScript 中用于声明变量的关键字之一，它是 ES6（ECMAScript 2015）引入的新特性之一。相比于旧版本的 var 关键字，let 具有更好的作用域规则和变量声明方式。 const constant 声明一个常量

**使用 let 声明的变量具有以下特点：

  1. 块级作用域：let 声明的变量具有块级作用域，即在 {} 内声明的变量只在该块内有效，超出块范围就无法访问。

    javascript function example() { if (true) { let x = 10; console.log(x); // 可以访问 x } console.log(x); // 报错，x 在块外不可见 } example();

  2. 不存在变量提升：使用 let 声明的变量不会发生变量提升，必须在声明后才能访问。

    javascript console.log(y); // 报错，y 未声明 let y = 20;

  3. 不允许重复声明：在同一个作用域内不允许重复声明同名变量，否则会报错。

    javascript let z = 30; let z = 40; // 报错，重复声明变量 z

  4. 全局作用域：在全局作用域中使用 let 声明的变量不会成为全局对象的属性。

    javascript let globalVar = 50; console.log(window.globalVar); // undefined，全局对象中不存在 globalVar

let 的引入弥补了 var 的一些缺陷，使得 JavaScript 的变量声明和作用域规则更加清晰和可控。因此，现代 JavaScript 开发中推荐优先使用 let 来声明变量。

响应式布局常用代码

rem - 缩写的英文全称：Root Em - 代码功能：rem 是 CSS 中的一个长度单位。它代表相对于根元素（通常是 <html> 元素）的字体大小来计算的相对单位。例如，如果根元素的字体大小是 16px，那么 1rem 就等于 16px。使用 rem 有助于创建响应式和可访问的设计，因为改变根元素的字体大小会相应地调整所有使用 rem 的元素的大小。

事件绑定常用代码

常用的事件字符串是指在使用 addEventListener 方法时常见的事件类型字符串。以下是一些常用的事件字符串：

  1. 鼠标事件：

    • click：单击鼠标左键时触发。
    • dblclick：双击鼠标左键时触发。
    • mousedown：鼠标按下时触发。
    • mouseup：鼠标松开时触发。
    • mousemove：鼠标移动时触发。
    • mouseover：鼠标移入元素时触发。
    • mouseout：鼠标移出元素时触发。
    • mouseenter：鼠标进入元素时触发（不冒泡）。
    • mouseleave：鼠标离开元素时触发（不冒泡）。

    • contextmenu：右键点击时触发上下文菜单事件。

  2. 键盘事件：

    • keydown：按下键盘上的任意键时触发。
    • keyup：释放键盘上的键时触发。

    • keypress：在按下并释放键盘上的键时触发（不常用）。

  3. 表单事件：

    • submit：提交表单时触发。
    • reset：重置表单时触发。
    • input：输入框内容改变时触发（包括输入、删除、粘贴等）。

    • change：表单元素的值改变时触发（通常用于 <input>、<select>、<textarea> 等）。

  4. 窗口和文档事件：

    • load：文档或资源加载完成时触发。
    • unload：窗口或框架被卸载时触发。
    • resize：窗口大小改变时触发。

    • scroll：元素滚动时触发。

  5. 焦点事件：

    • focus：元素获得焦点时触发。
    • blur：元素失去焦点时触发。
    • focusin：元素获得焦点时触发（冒泡）。
    • focusout：元素失去焦点时触发（冒泡）。

这些事件字符串用于指定要监听的事件类型，以便在元素触发相应的事件时执行相应的事件处理函数。可以根据需要选择合适的事件字符串来绑定事件。
